# ⚡ Orbit: Solid-State Battery Degradation Analytics & AI Agent

An end-to-end Machine Learning (ML) and Autonomous AI Agent project built for analyzing, predicting, and optimizing the cycle life and degradation profiles of solid-state lithium battery cells.

This project integrates physics-inspired data generation, predictive scikit-learn models, an autonomous reasoning agent (using a Plan-Act-Sense ReAct cycle), and a premium dark-themed Streamlit analytics dashboard.

---

## 🔋 Project Features

1. **Physics-Driven Degradation Simulator**: Simulates capacity retention (%) and interfacial resistance (Ohm) decay curves under varying environmental (temperature, pressure) and design parameters (electrolyte type, thickness, charge/discharge rates).
2. **Machine Learning Pipeline**: Trains dual Random Forest Regressors to predict cell degradation outcomes with high accuracy ($R^2 > 0.95$).
3. **Autonomous ReAct Agent (OrbitAgent)**: Employs a loop of planning, acting, and sensing. Equipped with a **Wikipedia Search Tool** and a **Python Execution Sandbox** allowing the agent to write and execute python code locally to query the ML models and synthesize research reports.
4. **Interactive Streamlit Web App**: A beautiful, custom-styled dark-mode dashboard to run predictions in real-time, view degradation charts, run the autonomous agent live, inspect feature importances, and download reports.

---

## 📁 Repository Structure

- `ml_pipeline.py`: Contains data generation, model training, evaluation, saving (`battery_model.joblib`), and Kaggle mock submission generating.
- `agent.py`: Defins the `OrbitAgent` class with Wikipedia searching, local sandboxed execution, and optional Gemini LLM planning.
- `app.py`: The frontend dashboard written in Streamlit.
- `laiba-haroon.ipynb`: The complete documented Jupyter notebook showing the step-by-step EDA, training, and agent runs.
- `submission.csv`: Kaggle format test set predictions.
- `test.csv` & `battery_degradation_data.csv`: Datasets generated by the pipeline.
- `.env`: Stores environment keys (e.g. `GEMINI_API_KEY`).

## 🚀 How to Run

### 1. Launch the Analytics Web App (Recommended)
Launch the beautiful Streamlit web interface:
```bash
streamlit run app.py
```
This will spin up a local dev server (usually at `http://localhost:8501`) where you can:
- Use interactive sliders to visualize decay curves.
- Input natural language questions and watch the Orbit Agent think and run code in real-time.
- Retrain the model pipeline or generate Kaggle submissions.

### 2. Run the Autonomous Agent (CLI)
To run a standalone agent reasoning loop in your terminal:
```bash
python agent.py
```

### 3. Run the ML Training Pipeline
To generate a new dataset and retrain the Random Forest regressors:
```bash
python ml_pipeline.py
```

### 4. Interactive Jupyter Notebook
Open `laiba-haroon.ipynb` in VS Code or Jupyter Lab to view the documented code walkthrough, plots, and agent outputs step-by-step.
